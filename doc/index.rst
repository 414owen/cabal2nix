This site provides documentation for the `Haskell <http://haskell.org/>`_
infrastructure provided by the `Nix Packages Collection <https://nixos.org/>`_.
The source code for all those documents featured here is `publicly available in
a Git Repository <https://github.com/NixOS/cabal2nix/tree/master/doc>`_, so
please don't hesitate to suggest edits, to add contents, or to point out errors
and omissions.
